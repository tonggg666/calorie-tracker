# 🥗 轻食记 Calorie.AI (v3.0)

本项目是一个基于 Web 的轻量级卡路里追踪工具，致力于提供简洁、专业且高度自定义的饮食记录体验。所有数据存储在本地浏览器，保障您的隐私。

## ✨ 核心特点

* **专业 UI/UX:** 采用现代化的 Slate/Teal 配色，界面简洁美观。
* **高级交互:** 基于 Tab 切换的餐食记录界面，移动端体验优化。
* **智能估算:** 基于食物关键词、份量和油腻程度进行热量模糊估算。
* **科学目标:** 基于 Mifflin-St Jeor 公式自动计算 TDEE，提供个性化减脂目标。
* **数据持久化:** 使用 `localStorage` 技术，本地保存所有历史记录和身体档案。

## 🛠️ 技术栈

本项目为纯前端静态网站，易于部署和使用。

* **结构:** HTML5
* **样式:** Tailwind CSS (通过 CDN 引入)
* **逻辑:** Vanilla JavaScript

## 🚀 如何运行和部署

### 本地运行

1.  将 `index.html` 文件下载到本地。
2.  直接用任何现代浏览器（Chrome, Firefox, Edge, Safari）打开该文件即可使用。

### 公网部署 (推荐)

本项目是完美的静态网站。你可以使用 **GitHub Pages** 或 **Netlify** 等免费服务快速部署。

1.  将本项目 Fork 或 Clone 到你的 GitHub 账号。
2.  确保主文件名为 `index.html`。
3.  在仓库设置中启用 GitHub Pages。
4.  你的应用即可通过公网链接访问！
